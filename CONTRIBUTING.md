# Contributing to Guardian Override

Thank you for your interest in contributing! Guardian Override is built to help families set healthy screen boundaries with dignity and respect. We welcome contributions that strengthen clarity, usability, and trust.

---

## How you can help

- **Report issues:** Share bugs, usability problems, or unclear documentation in the Issues tab.
- **Suggest improvements:** Propose new features, rule schemas, or user flows that align with our mission.
- **Improve documentation:** Help make guides, onboarding, and explanations easier to understand.
- **Write code:** Contribute to modules like the simulator, schemas, extensions, or mobile scaffolds.
- **Review pull requests:** Give constructive feedback that is specific, kind, and focused on ideas.

---

## Getting started

**1. Fork the repository and create your branch from `main`.**

**2. Install dependencies:**
```
npm install
```
**3. Run the simulator:**
```
npm start
```
**4. Test your changes before submitting a pull request.**

**5. Write clear commit messages that explain the change.**

---

## Contribution standards

**Respect:** 
Treat all collaborators with empathy and professionalism.

**Clarity:** Use plain language in code comments, documentation, and discussions.

**Transparency:** Document assumptions and decisions so others can follow your work.

**Consistency:** Follow existing file structures and naming conventions.

**Accessibility:** Keep user experience simple and inclusive.

---

## Code of Conduct

By contributing, you agree to follow the Community Code of Conduct. This ensures our community stays safe, welcoming, and constructive.

---

## Pull request process

- Open a pull request with a clear description of what you changed and why.

- Link related issues if applicable.

**Expect review:** Maintainers may request changes or clarifications.

**Approval:** Once approved, your changes will be merged into main.

---

## Questions and support

**If you’re unsure about something:**

- Open a discussion in the [Discussions](https://github.com/litalien101/guardian-override/discussions) tab.

- Reach out to maintainers through the designated contact channel.

---

## Thank you
**Every contribution (whether code, documentation, or feedback) helps Guardian Override grow into a tool families can trust. We’re grateful for your time and effort.**
